## Getting Started

Simply open `index.html` in your web browser. No build process required!

## Categories

The app includes pre-configured expense and income categories matching your existing Google Sheets setup.

### Income Categories
- Paycheck
- Other income

### Expense Categories
- Housing, Transportation, Suministros, Grocery, Restaurants
- Clothing, Subscription, Desarrollo personal
- Otros gastos personales, Gifts, Entertainment
- Vacation, Drinks/Tapas, Party, Efectivo
- Planes finde, Golf, Impuestos/multas, Deporte, Glovo

## Data Storage

Currently, all data is stored locally in your browser's localStorage. Cloud sync functionality is planned for future updates.

## Export

You can export all your transactions to CSV format from the History tab.
